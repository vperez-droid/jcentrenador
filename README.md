# JCT - App de Entrenamiento Personal

Esta es una aplicación web desarrollada con Streamlit para la gestión de entrenamientos entre un entrenador y sus clientes.

## Características

- **Panel de Entrenador:** Permite registrar clientes y asignar rutinas.
- **Panel de Cliente:** Permite a los clientes ver sus rutinas, historial y progreso.

## Instalación

1.  Clona este repositorio:
    ```
    git clone [URL-de-tu-repo]
    ```
2.  Crea un entorno virtual e instálalo:
    ```
    python -m venv venv
    source venv/bin/activate  # En Windows: venv\Scripts\activate
    ```
3.  Instala las dependencias:
    ```
    pip install -r requirements.txt
    ```
